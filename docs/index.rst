Messages v0.5
====================================

Welcome to the **messages** docs!

* Messages is a python3 **Open Source** package designed to make sending messages easy.
* This package will encapsulate various standard library, third-party packages, web API calls, etc. all in one package for easy use with a **standard API across all message-types**.
* This project is intended to be **community-driven** and a great way to learn about software engineering processes.
* All contributors are welcome regardless of experience level!

Join the conversation on our `Slack Team <https://messages-py.herokuapp.com/>`_
